# Active Directory Labs

This directory contains Active Directory security labs, investigations, and defensive analysis.

Topics covered:
- Kerberos attacks
- LDAP enumeration
- BloodHound analysis
- Privilege escalation
- Lateral movement
- Group Policy abuse
- Windows Event Logs

Tools:
- BloodHound
- Sysmon
- Event Viewer
- PowerShell
- Splunk
- Wazuh

Platforms:
- TryHackMe
- Hack The Box
- Blue Team Labs Online

Focus:
Understanding how attackers abuse Windows environments and how defenders detect suspicious activity.

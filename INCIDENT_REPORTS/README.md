# Incident Reports

This directory contains simulated incident response investigations and case reports.

Each report may include:
- Executive summary
- Attack timeline
- Indicators of compromise (IOCs)
- Detection methods
- Containment actions
- Lessons learned

Platforms:
- LetsDefend
- CyberDefenders
- Blue Team Labs Online
- TryHackMe

Goal:
Practicing real-world SOC investigation and reporting workflows.
